# softqlearning-pytorch

## Installation

Requirements:
* Anaconda >= 24.1.2
* Python >= 3.9
* PyTorch >= 2.2.1
* MuJoCo >= 3.1.3

```
conda env create -f environment.yml
```

## Usage

```
python train.py
```
